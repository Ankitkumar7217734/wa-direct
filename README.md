# WA Direct

Open any WhatsApp chat instantly — no need to save the contact first.

A single-page web app: type a phone number (with country code), optionally a message, and it opens `wa.me/<number>?text=<message>` for you. Recent numbers are kept in `localStorage`; nothing is sent to a server.

## Live demo

<https://ankitkumar7217734.github.io/wa-direct/>

## Use it locally

Just open `index.html` in any browser. No build step, no dependencies.

## Features

- Country code picker
- Pre-filled message support
- Copy link button
- Recent numbers (stored locally)
- Installable as a PWA
- Dark theme
